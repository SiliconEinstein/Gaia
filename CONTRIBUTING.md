# Contributing

Contributor and agent rules for this repository live in [`AGENTS.md`](AGENTS.md), which is the
canonical guide for both human contributors and in-repo agents (Claude Code, Cursor, Codex, …).

For everything below, read `AGENTS.md`:

- Local setup and installing the git hooks (`make bootstrap`).
- Quality gates (`pre-commit` + `pre-push` + `commit-msg`).
- Commit format (Conventional Commits).
- Code style, doc fidelity, protected layers, scripts logging template.

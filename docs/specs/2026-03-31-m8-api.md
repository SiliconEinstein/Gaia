# M8 — HTTP API Spec

> **Status:** Draft
> **Date:** 2026-03-31
> **实现文件:** `gaia_lkm/lkm/services/`

## 概述

M8 是 FastAPI 应用，暴露读取端 + ingest 触发 + admin 端点。依赖 M2（StorageManager）+ M5（integrate）。M6/M7 为可选（admin 端点触发）。

## 上游参考

- `docs/foundations/lkm/06-api.md`：API 设计

---

## 应用架构

```python
# app.py
app = FastAPI(lifespan=lifespan)

# lifespan: 启动时初始化 StorageManager，关闭时清理
async def lifespan(app: FastAPI):
    app.state.storage = await StorageManager.create(config)
    yield
    await app.state.storage.close()

# 依赖注入
def get_storage(request: Request) -> StorageManager:
    return request.app.state.storage
```

## 端点

### Ingest

#### `POST /packages/ingest`

触发 Pipeline A：接收 Gaia IR package，执行 lower → integrate 全流程。

**请求体**：

```python
class IngestRequest(BaseModel):
    package: dict          # LocalCanonicalGraph 序列化
    review_reports: list[dict]
    package_id: str
    version: str
```

**响应**：

```python
class IngestResponse(BaseModel):
    package_id: str
    status: str            # "merged" | "failed"
    variable_count: int    # 新增/已存在的 variable 数
    factor_count: int      # 新增/已存在的 factor 数
    new_variables: int     # 本次新建的 global variable 数
    new_factors: int       # 本次新建的 global factor 数
    unresolved_cross_refs: list[dict]
```

**内部流程**：
1. 反序列化 `LocalCanonicalGraph`（`gaia.gaia_ir` 上游模型）
2. Pipeline A lowering（M3）
3. Integrate（M5）
4. 返回结果

### Variables

#### `GET /variables`

列出 variable nodes，支持分页 + BM25 搜索。

**查询参数**：

| 参数 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `q` | str \| None | None | BM25 全文搜索（匹配 local variable 的 content） |
| `type` | str \| None | None | 按 type 过滤（`claim` / `setting` / `question`） |
| `visibility` | str | `"public"` | 按 visibility 过滤 |
| `limit` | int | 20 | 分页大小 |
| `offset` | int | 0 | 分页偏移 |

**响应**：

```python
class VariableListResponse(BaseModel):
    items: list[VariableDetail]
    total: int
    limit: int
    offset: int
```

#### `GET /variables/{id}`

查询单个 variable node（by gcn_id）。

**响应**：

```python
class VariableDetail(BaseModel):
    id: str                         # gcn_id
    type: str
    visibility: str
    content: str | None             # 通过 representative_lcn join 获取
    content_hash: str
    parameters: list[dict]
    representative_lcn: dict
    local_members: list[dict]
    belief: float | None            # 最新 BeliefSnapshot 中的值（如有）
    metadata: dict | None
```

**content 获取路径**：
```
global_variable_nodes[gcn_id].representative_lcn.local_id
  → local_variable_nodes[local_id].content
```
两次主键查询，不 join。

**错误**：`404` — variable 不存在。

### Factors

#### `GET /factors/{id}`

查询单个 factor node（by gfac_id）。

**响应**：

```python
class FactorDetail(BaseModel):
    id: str                         # gfac_id
    factor_type: str
    subtype: str
    premises: list[str]             # gcn_ids
    conclusion: str                 # gcn_id
    representative_lfn: str
    steps: list[dict] | None       # 通过 representative_lfn join 获取
    source_package: str
    metadata: dict | None
```

**steps 获取路径**：
```
global_factor_nodes[gfac_id].representative_lfn
  → local_factor_nodes[lfn_id].steps
```

**错误**：`404` — factor 不存在。

---

### Graph

#### `GET /graph/subgraph`

N 跳子图查询。

**查询参数**：

| 参数 | 类型 | 默认值 | 说明 |
|---|---|---|---|
| `node_id` | str | required | 起始 gcn_id |
| `hops` | int | 1 | 跳数 |
| `direction` | str | `"both"` | `upstream` / `downstream` / `both` |

**响应**：

```python
class SubgraphResponse(BaseModel):
    variables: list[VariableDetail]
    factors: list[FactorDetail]
    edges: list[dict]               # {source, target, role: "premise"|"conclusion"}
```

**实现**：
- 有 Neo4j/Kuzu → 用图查询（高效）
- 无图存储 → 从 LanceDB factor 表过滤（性能可接受，因为按 gcn_id 索引）

### Beliefs

#### `GET /beliefs/snapshots`

BeliefSnapshot 列表（按 timestamp 降序）。

**查询参数**：`limit`, `offset`。

#### `GET /beliefs/snapshots/{snapshot_id}`

单个 BeliefSnapshot 详情。

#### `GET /beliefs/variables/{id}`

某个 variable 的 belief 历史（跨多个 snapshot）。

---

### Admin

#### `POST /curation/run`

手动触发 curation discovery（M6）。

**响应**：CurationReport 或异步 task ID。

#### `POST /bp/run`

手动触发 global BP（M7）。

**请求体**（可选）：

```python
class BPRunRequest(BaseModel):
    resolution_policy: str = "latest"
    prior_cutoff: datetime | None = None  # None = 当前时间
```

**响应**：BeliefSnapshot 或异步 task ID。

## 错误处理

| 状态码 | 场景 |
|---|---|
| `404` | 资源未找到（variable、factor、snapshot） |
| `422` | 请求参数验证失败（Pydantic validation error） |
| `503` | 存储未初始化（启动失败或缺少配置） |

---

## 关键约束

1. **读取端点不修改状态**：GET 端点是纯查询
2. **Content 通过 representative_lcn 间接获取**：不在 global 层存 content
3. **Steps 通过 representative_lfn 间接获取**：不在 global 层存 steps
4. **默认仅返回 public variable**：除非显式指定 `visibility=private`
5. **无认证**：仅供内部/开发使用
